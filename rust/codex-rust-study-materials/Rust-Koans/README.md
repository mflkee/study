# Rust Koans

Здесь собраны короткие тесты по базовым темам Rust. Логика та же, что в
`JS-Koans` из исходного репозитория:

- открываете тестовый файл;
- находите `todo!()`;
- заменяете его правильным значением или выражением;
- запускаете тесты до полного прохождения файла.

## Порядок прохождения

1. `01_basics.rs`
2. `02_ownership.rs`
3. `03_structs_enums.rs`
4. `04_collections_iterators.rs`
5. `05_results_options.rs`
6. `06_traits_lifetimes.rs`
7. `07_pattern_matching.rs`
8. `08_strings_utf8.rs`
9. `09_threads_channels.rs`

## Команды

Запустить конкретную тему:

```bash
cargo test --test 01_basics
```

Запустить все темы:

```bash
cargo test
```

## Правило

Сначала пытайтесь понять, почему тест падает, и только потом подставляйте ответ.
Koans здесь нужны не для угадывания значений, а для фиксации модели мышления
Rust.
