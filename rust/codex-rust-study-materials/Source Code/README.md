# Исходники

Этот раздел играет ту же роль, что и `Исходники` в исходном репозитории:

- показать готовую структуру проекта;
- дать студенту точку опоры перед самостоятельной сборкой своего решения;
- продемонстрировать, как выглядят модули, тесты и организация Rust-кода в
  небольшом проекте.

Темы:

- `Тема 4.1. Рабочее окружение современного Rust-проекта`
- `Тема 4.2. Модули и workspace`
- `Тема 4.3. Тестирование проекта`
