# Source Code

Этот раздел играет ту же роль, что и `Source Code` в исходном репозитории:

- показать готовую структуру проекта;
- дать студенту точку опоры перед самостоятельной сборкой своего решения;
- продемонстрировать, как выглядят модули, тесты и организация Rust-кода в
  небольшом проекте.

Темы:

- `Topic 4.1. Modern Rust Project Environment`
- `Topic 4.2. Modules and Workspace`
- `Topic 4.3. Project Testing`
