# Учебный материал по Rust

- работа с ownership и borrowing;
- проектирование через `struct`, `enum`, `trait` и композицию вместо
  классического наследования;
- обработка ошибок через `Option` и `Result`;
- коллекции, итераторы и замыкания;
- файловый ввод-вывод;
- консольные приложения;
- модули, crates, workspace;
- тестирование, документация и качество кода;
- базовый безопасный параллелизм через threads и channels;
- тесты, модули, workspace и организация проекта.

## Структура каталога

- `Theory` - подробный конспект по темам курса с кодом, вопросами и ссылками на
  задания.
- `Rust-Koans` - набор небольших тестов для закрепления синтаксиса и базовых
  концепций.
- `Practice` - короткие задания на один основной приём.
- `Homework` - более крупные задачи с заготовками и тестами.
- `Source Code` - готовые примеры по темам окружения, модулей и тестирования.
- `Working in the Terminal and Cargo.md` - минимальный гайд по командам, которые
  понадобятся студенту.
- `Rust Code Style Guide.md` - базовый кодстайл и инженерные правила.

   `3.2`, `3.4`.

## Как работать с материалом

Почти все упражнения специально оставлены с незавершённой логикой:

- в `Rust-Koans` студент меняет `placeholder::<...>()` внутри тестов;
- в практиках студент дописывает `src/main.rs`;
- в домашних заданиях студент дописывает `src/lib.rs`, `src/main.rs` и
  добивается прохождения тестов.

Такой формат ближе к исходному репозиторию: не просто читать конспект, а
проходить через обязательное написание кода.

## Как запускать

У этого каталога нет общего workspace. Это сделано специально, чтобы каждую
задачу можно было открывать и проверять изолированно.

Примеры:

```bash
cd ~/study/rust/codex-rust-study-materials/Rust-Koans
cargo test --test 01_basics

cd ~/study/rust/codex-rust-study-materials/Homework/1.4\ Iterators\ and\ Closures\ in\ Rust/starter
cargo test

cd ~/study/rust/codex-rust-study-materials/Practice/8\ Mini\ WC\ Utility/starter
cargo run -- README.md
```

## Что студент должен уметь к финалу

После прохождения курса студент должен уметь:

- создать и настроить Rust-проект через `cargo`;
- объяснить ownership, borrowing, slices и lifetimes на базовом уровне;
- проектировать модели предметной области через `struct`, `enum` и `trait`;
- использовать `Vec`, `HashMap`, строки, итераторы и методы коллекций;
- понимать UTF-8 специфику строк в Rust;
- проектировать обобщённый код через traits и generics;
- писать код без постоянного `unwrap`;
- читать и писать файлы;
- собирать CLI-программы;
- писать unit, integration и doc tests;
- разнести проект по модулям и при необходимости собрать workspace;
- понимать базовые примитивы конкурентности: `thread`, `mpsc`, `Arc`, `Mutex`;
- выполнить самостоятельный учебный проект на Rust без шаблонного копирования.
