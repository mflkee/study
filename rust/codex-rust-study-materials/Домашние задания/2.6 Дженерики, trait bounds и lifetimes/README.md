# Домашнее задание к теме «Дженерики, trait bounds и lifetimes»

## Задача. Универсальный рейтинг учебных результатов

Нужно написать обобщённый код, который умеет работать с разными типами учебных результатов.

В системе есть:

- результаты домашних работ;
- результаты экзаменов.

У них разные поля, но общее у них одно: у каждого результата есть название и числовой балл.

## Что нужно сделать

1. Реализовать `trait Scorable`.
2. Реализовать структуры:
   - `HomeworkResult`;
   - `ExamResult`.
3. Реализовать `Scorable` для обеих структур.
4. Реализовать generic-функции:
   - `best_item`;
   - `average_score`;
   - `top_titles`.
5. Реализовать функцию `longer_title`, которая возвращает более длинную строку по ссылке.

## На что обратить внимание

- Не копируйте алгоритмы для разных типов, если можно обобщить через trait.
- Если функция возвращает ссылку, подумайте о времени жизни этой ссылки.
- Не делайте лишних `String`, если можно вернуть `&str`.
