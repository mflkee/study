# Домашнее задание к теме «Option, Result и обработка ошибок»

## Задача. Планировщик учебного дня

Нужно реализовать небольшой планировщик занятий. Входные данные приходят в
строковом формате:

```text
09:30|Прочитать главу про ownership|high
```

Формат:

- время `HH:MM`;
- название задачи;
- приоритет `low`, `medium`, `high`.

## Что нужно сделать

1. Реализуйте `enum Priority`.
2. Реализуйте `struct StudyTask`.
3. Реализуйте `enum ScheduleError`.
4. Реализуйте функцию `parse_task`.
5. Реализуйте структуру `StudySchedule` и методы:
   - `new`;
   - `add_from_line`;
   - `remove_by_time`;
   - `tasks_for_priority`;
   - `next_after`;
   - `all`.

## На что обратить внимание

- Не используйте `unwrap` для пользовательского ввода.
- Неверный формат должен приводить к `Err`, а не к `panic!`.
- У `StudySchedule` не должно быть двух задач на одно и то же время.
