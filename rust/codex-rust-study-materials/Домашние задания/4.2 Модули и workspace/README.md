# Домашнее задание к теме «Модули и workspace»

## Задача

Возьмите проект из предыдущего задания и разнесите его по модулям. Если логика
уже стала достаточно большой, соберите `workspace` из двух crates:

- `core` - библиотечная логика;
- `cli` - консольный вход в приложение.

## Что нужно сделать

1. Вынести доменную модель в отдельный модуль.
2. Вынести работу с хранением данных в отдельный модуль.
3. Вынести форматирование вывода в отдельный модуль.
4. При желании разделить проект на два crates через workspace.
5. Убедиться, что публичный API библиотеки остаётся понятным.

## Важно

Не дробите проект на модули искусственно. Разделение должно отражать
ответственность кода, а не количество файлов.
