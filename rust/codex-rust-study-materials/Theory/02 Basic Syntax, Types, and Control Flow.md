# 02 Basic Syntax, Types, and Control Flow

## Главная мысль темы

Rust внешне напоминает C-подобный язык, но важны не скобки, а то, что почти всё
в нём выражено явно: типы, изменяемость, возвращаемые значения, ветвления.

## Переменные и `mut`

По умолчанию переменные неизменяемы:

```rust
let score = 5;
let mut total = 0;
total += score;
```

Это хороший дефолт: код с меньшим количеством изменяемого состояния проще
понимать и тестировать.

## Базовые типы

- целые: `i32`, `u8`, `usize`;
- дробные: `f32`, `f64`;
- логический: `bool`;
- символ: `char`;
- строковый срез: `&str`.

Нужно особенно различать `String` и `&str`. Это одна из первых “развилок
мышления” в Rust.

## Управление потоком

`if` в Rust это выражение:

```rust
let label = if score >= 60 { "pass" } else { "fail" };
```

`loop` тоже может возвращать значение:

```rust
let result = loop {
    break 42;
};
```

Есть и привычные `while`, `for`, но в Rust часто лучше использовать итераторы
там, где код от этого становится понятнее.

## Массивы и кортежи

Массив фиксированной длины:

```rust
let numbers = [1, 2, 3];
```

Кортеж для небольших связанных значений:

```rust
let student = ("Mira", 5, true);
```

Если структура начинает жить долго или иметь смысл как сущность, лучше перейти к
`struct`.

## Частые ошибки

- Забывать, что `if` возвращает значение, и писать лишние `return`.
- Использовать `mut` “на всякий случай”.
- Путать `char` и `&str`.
- Выбирать типы чисел случайно, не думая про диапазон и смысл.
