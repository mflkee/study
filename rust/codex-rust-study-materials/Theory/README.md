книге, а локальный конспект, который ведёт по темам курса и сразу отправляет в
нужные упражнения.

## Как пользоваться

На каждой теме есть:

- краткая теория;
- куски кода;
- типичные ошибки;
- прямые ссылки на koans, практику и домашние задания;
- вопросы для самопроверки.

Лучший режим работы:

1. Прочитать тему.
2. Сразу сделать koans или практику по ней.
3. Вернуться к теме и перечитать блок “частые ошибки”.
4. Потом брать домашнее задание.
