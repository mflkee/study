# 14 Параллельный подсчёт

## Описание задания

Сделайте программу, которая параллельно обрабатывает набор чисел.

Минимальный вариант:

- разбить числа на чанки;
- запустить несколько потоков;
- собрать частичные суммы;
- вывести общий результат.

## Процесс реализации

1. Подготовьте входной список чисел.
2. Разбейте его на части.
3. Для каждой части запустите отдельный поток.
4. Дождитесь завершения потоков и сложите результаты.

### Повышенный уровень сложности

1. Используйте `mpsc::channel`, чтобы потоки возвращали результаты.
2. Сравните последовательную и параллельную реализацию.
