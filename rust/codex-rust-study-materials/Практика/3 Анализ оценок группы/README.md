## Описание задания

Соберите программу, которая анализирует список оценок студентов и выводит:

- средний балл;
- минимальную и максимальную оценку;
- количество сдавших и не сдавших.

Пример входа:

`cargo run -- 5 4 3 5 2 4`

## Процесс реализации

1. Прочитайте все оценки из аргументов.
2. Преобразуйте строки в числа.
3. С помощью `Vec` и циклов или итераторов вычислите статистику.
4. Выведите результат в читаемом виде.

### Повышенный уровень сложности

1. Считайте проходным баллом не только `3`, но и значение из отдельного аргумента.
2. Выведите распределение оценок по частотам.
