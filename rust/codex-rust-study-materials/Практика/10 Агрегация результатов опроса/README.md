## Описание задания

Напишите программу, которая агрегирует результаты опроса из текстового файла.

Предлагаемый формат файла:

```text
Rust;12
Go;4
Python;9
Rust;3
```

## Процесс реализации

1. Прочитайте файл.
2. Разберите каждую строку на вариант ответа и количество голосов.
3. Сложите одинаковые варианты.
4. Выведите итоговую таблицу.

### Повышенный уровень сложности

1. Отсортируйте ответы по убыванию голосов.
2. Выведите победителя и долю в процентах.
