## Описание задания

Напишите программу, которая считает, сколько раз каждое слово встречается в тексте.

Это упражнение нужно, чтобы закрепить `HashMap`, строки и итераторы.

## Процесс реализации

1. Получите текст из аргумента командной строки или из строки в коде.
2. Приведите текст к одному регистру.
3. Разбейте строку на слова.
4. Подсчитайте частоты через `HashMap<String, usize>`.
5. Выведите результат.

### Повышенный уровень сложности

1. Отсортируйте слова по частоте.
2. Игнорируйте знаки препинания.
