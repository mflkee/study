# 4 Каталог книг

## Описание задания

Реализуйте небольшой каталог книг на основе `struct`.

Для каждой книги храните:

- название;
- автора;
- год;
- количество страниц.

## Процесс реализации

1. Создайте `struct Book`.
2. Создайте `Vec<Book>` с несколькими книгами.
3. Напишите функции поиска по автору и фильтрации по году.
4. Выведите найденные книги в консоль.

### Повышенный уровень сложности

1. Добавьте метод `is_large`, который определяет, является ли книга большой.
2. Реализуйте отдельную функцию сортировки по году.
