# 8 Mini WC Utility

## Описание задания

Напишите упрощённую версию утилиты `wc`, которая считает:

- количество строк;
- количество слов;
- количество байт в файле.

## Процесс реализации

1. Получите путь к файлу из аргументов командной строки.
2. Прочитайте файл в строку.
3. Посчитайте строки, слова и байты.
4. Выведите статистику.

### Повышенный уровень сложности

1. Поддержите флаги `--lines`, `--words`, `--bytes`.
2. Для ошибок чтения возвращайте понятное сообщение без `panic!`.
