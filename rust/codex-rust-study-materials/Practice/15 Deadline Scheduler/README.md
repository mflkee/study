# 15 Deadline Scheduler

## Описание задания

Реализуйте небольшой планировщик дедлайнов.

Формат записи:

```text
2026-03-12|Подготовить отчёт|high
```

## Процесс реализации

1. Опишите задачу через `struct`.
2. Приоритет задайте через `enum`.
3. Реализуйте загрузку списка задач из файла.
4. Отсортируйте задачи по дате.
5. Выведите ближайшие дедлайны.

### Повышенный уровень сложности

1. Добавьте фильтрацию только просроченных задач.
2. Добавьте подсчёт количества задач по приоритетам.
