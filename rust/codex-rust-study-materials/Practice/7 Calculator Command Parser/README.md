# 7 Парсер команд калькулятора

## Описание задания

Реализуйте парсер строковых команд для простого калькулятора.

Поддержите форматы:

- `add 2 3`
- `sub 10 4`
- `mul 6 7`
- `div 20 5`

## Процесс реализации

1. Создайте `enum Command`.
2. Напишите функцию разбора строки в `Command`.
3. Напишите функцию исполнения команды.
4. Корректно обрабатывайте деление на ноль.

### Повышенный уровень сложности

1. Возвращайте свои ошибки через `Result`.
2. Поддержите отрицательные числа.
