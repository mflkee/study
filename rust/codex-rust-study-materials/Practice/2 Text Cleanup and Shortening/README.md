# 2 Text Cleanup and Shortening

## Описание задания

Напишите программу, которая принимает строку, убирает лишние пробелы, считает
длину очищенного текста и при необходимости сокращает его до заданного лимита.

Пример:

`cargo run -- "  Rust   ownership   and borrowing  " 12`

## Процесс реализации

1. Прочитайте исходную строку и лимит.
2. Удалите лишние пробелы между словами.
3. Выведите очищенный текст и его длину.
4. Если длина больше лимита, выведите сокращённую версию.

### Повышенный уровень сложности

1. Сохраняйте слова целиком и не обрезайте строку посередине слова.
2. Добавьте суффикс `...` при сокращении.
