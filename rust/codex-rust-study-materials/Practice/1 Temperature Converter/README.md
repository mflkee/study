# 1 Temperature Converter

## Описание задания

Напишите консольную программу, которая переводит температуру между шкалами
Цельсия и Фаренгейта.

Примеры:

- `cargo run -- c2f 20`
- `cargo run -- f2c 68`

## Процесс реализации

1. Получите аргументы командной строки через `std::env::args`.
2. Проверьте режим работы: `c2f` или `f2c`.
3. Преобразуйте строковое значение температуры в число.
4. Выведите результат с понятным сообщением.

### Повышенный уровень сложности

1. Добавьте округление до двух знаков после запятой.
2. Добавьте обработку неверного режима без `panic!`.
