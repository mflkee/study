# 12 Мини-поисковик по заметкам

## Описание задания

Сделайте мини-поисковик по текстовым заметкам.

Программа получает:

- путь к файлу;
- поисковый запрос.

Дальше она должна вывести строки, в которых встречается запрос.

## Процесс реализации

1. Получите аргументы командной строки.
2. Прочитайте файл построчно или целиком.
3. Реализуйте поиск подстроки.
4. Выведите подходящие строки с номерами.

### Повышенный уровень сложности

1. Добавьте режим нечувствительности к регистру.
2. Выведите общее число найденных совпадений.
