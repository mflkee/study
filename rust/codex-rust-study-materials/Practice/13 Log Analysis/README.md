# 13 Log Analysis

## Описание задания

Напишите программу, которая анализирует лог-файл и считает:

- сколько было `INFO`, `WARN`, `ERROR`;
- какие модули чаще всего пишут ошибки;
- последнее сообщение для выбранного модуля.

Предлагаемый формат строки:

```text
ERROR|auth|invalid token
```

## Процесс реализации

1. Прочитайте файл.
2. Разберите каждую строку на уровень, модуль и сообщение.
3. Подсчитайте статистику через `HashMap`.
4. Выведите сводку.

### Повышенный уровень сложности

1. Отбрасывайте битые строки и отдельно считайте их количество.
2. Добавьте фильтр по модулю из аргументов CLI.
