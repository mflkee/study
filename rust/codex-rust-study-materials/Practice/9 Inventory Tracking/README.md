# 9 Inventory Tracking

## Описание задания

Соберите простую модель склада.

Нужно уметь:

- приходовать товар;
- списывать товар;
- печатать текущий остаток.

## Процесс реализации

1. Создайте `struct Item` или используйте `HashMap<String, i32>`.
2. Опишите операции прихода и списания.
3. Запретите уход в отрицательный остаток.
4. Выведите итоговое состояние склада.

### Повышенный уровень сложности

1. Добавьте тип товара через `enum`.
2. Выведите товары, которые нужно срочно пополнить.
