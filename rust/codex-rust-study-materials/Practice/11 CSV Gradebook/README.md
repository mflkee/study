# 11 CSV-журнал успеваемости

## Описание задания

Напишите программу, которая читает CSV-подобный файл с оценками студентов и
строит краткий отчёт.

Формат строки:

```text
Имя;Предмет;Оценка
```

Пример:

```text
Мира;Rust;5
Олег;Алгоритмы;4
Мира;Алгоритмы;5
```

## Процесс реализации

1. Прочитайте файл.
2. Разберите строки на поля.
3. Посчитайте средний балл по каждому студенту.
4. Посчитайте средний балл по каждому предмету.
5. Выведите итоговый отчёт.

### Повышенный уровень сложности

1. Отсортируйте студентов по среднему баллу.
2. Выведите отличников по заданному порогу.
