# 5 TODO в памяти

## Описание задания

Соберите минимальный TODO-менеджер, который работает только в памяти программы.

Поддержите команды:

- `add <текст>`;
- `list`;
- `done <индекс>`.

## Процесс реализации

1. Смоделируйте задачу через `struct`.
2. Для статуса используйте `bool` или отдельный `enum`.
3. Обрабатывайте команды через `match`.
4. Выводите список задач с индексами.

### Повышенный уровень сложности

1. Добавьте команду `remove`.
2. Добавьте фильтр `list done` и `list active`.
