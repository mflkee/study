# Домашнее задание к теме «CLI и файловый ввод-вывод»

## Задача. Консольный TODO-менеджер с хранением в файле

Нужно написать консольное приложение, которое хранит задачи в текстовом файле.

Поддерживаемые команды:

- `add <текст>`;
- `list`;
- `done <индекс>`;
- `remove <индекс>`;
- `help`.

## Формат хранения

Используйте простой текстовый формат:

```text
0|Купить книгу по Rust
1|Подготовить итоговый проект
```

Где:

- `0` - задача не выполнена;
- `1` - задача выполнена.

## Что нужно сделать

1. Реализуйте `Command`.
2. Реализуйте `Task`.
3. Реализуйте `TodoError`.
4. Реализуйте разбор аргументов через `parse_command`.
5. Реализуйте чтение и запись задач из файла.
6. Реализуйте операции добавления, отметки выполнения и удаления.
7. Реализуйте форматирование списка для печати в консоль.

## Что важно

- Вся доменная логика должна лежать в `src/lib.rs`.
- `src/main.rs` должен быть тонкой оболочкой.
- Нельзя опираться на `panic!` для штатных ошибок ввода.
