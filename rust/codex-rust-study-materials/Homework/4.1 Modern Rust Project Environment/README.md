# Домашнее задание к теме «Рабочее окружение современного Rust-проекта»

## Задача

Создайте отдельный Rust-проект и настройте минимально здоровое рабочее
окружение.

## Что нужно сделать

1. Создать новый проект через `cargo new`.
2. Инициализировать Git-репозиторий.
3. Добавить `README.md` с кратким описанием проекта.
4. Организовать код так, чтобы команда `cargo check` проходила без ошибок.
5. Подключить ежедневный цикл качества:
   - `cargo fmt`;
   - `cargo clippy --all-targets --all-features -- -D warnings`;
   - `cargo test`;
   - `cargo doc`.
6. Добавить хотя бы одну функцию в `src/lib.rs` и хотя бы один unit test.

## Результат

На выходе у вас должен быть отдельный репозиторий, который:

- собирается;
- форматируется;
- проверяется линтером;
- содержит тесты;
- содержит короткое описание проекта.
