# Домашнее задание к теме «Параллельная обработка данных»

## Задача. Параллельный анализ набора чисел и строк

Нужно реализовать несколько функций, которые используют потоки для параллельной
обработки данных.

## Что нужно сделать

1. Реализовать `enum ParallelError`.
2. Реализовать функцию `split_into_chunks`.
3. Реализовать `parallel_sum`.
4. Реализовать `parallel_max`.
5. Реализовать `parallel_word_count`.

## Обязательное условие

Для `parallel_sum` и `parallel_max` используйте `thread::spawn`. Для передачи
результатов можно использовать либо `JoinHandle`, либо `mpsc::channel`.

## Важно

- Не допускайте `worker_count == 0`.
- Пустой вход должен обрабатываться явно.
- Перед тем как распараллеливать, сначала корректно разделите данные на чанки.

## Результат при правильном решении

Все тесты из `tests/parallel.rs` проходят успешно.

## Повышенный уровень сложности

1. Реализуйте отдельную функцию, которая считает частоты слов параллельно.
2. Сравните время последовательной и параллельной версии на больших входных
   данных.
