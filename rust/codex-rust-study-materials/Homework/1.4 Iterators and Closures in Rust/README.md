# Домашнее задание к теме «Итераторы и замыкания в Rust»

## Задача 1. Отбор студентов к итоговому проекту

Напишите функции для анализа учебной группы.

Используйте `Iterator`-подход там, где он делает код яснее:

- `filter`;
- `map`;
- `sum`;
- `flat_map`;
- `max`.

### Что нужно сделать

1. Реализуйте структуру `Student`.
2. Реализуйте функцию `select_for_final_project`, которая возвращает только тех
   студентов, которые:
   - закрыли минимум 5 модулей;
   - набрали средний балл не ниже порога;
   - уже сдали промежуточный проект.
3. Реализуйте функцию `total_completed_modules`.
4. Реализуйте функцию `adjust_scores`, которая принимает замыкание и применяет
   его ко всем оценкам.
5. Реализуйте функцию `flatten_attempts`, которая превращает список попыток в
   один плоский список.
6. Реализуйте функцию `best_score_per_student`, которая для каждого студента
   находит его лучшую попытку.

## Результат при правильном решении

Все тесты из `tests/homework.rs` проходят успешно.

## Требования к выполнению

1. Код проходит `cargo test`.
2. Код форматируется через `cargo fmt`.
3. Код не даёт предупреждений на `cargo clippy --all-targets --all-features --
   -D warnings`.

## Решение

1. Откройте проект `starter`.
2. Допишите `src/lib.rs`.
3. Запустите `cargo test`.

## Повышенный уровень сложности

1. Реализуйте сортировку результатов по убыванию среднего балла.
2. Сделайте отдельную функцию, которая возвращает только имена студентов,
   допущенных к финалу.
